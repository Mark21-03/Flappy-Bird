### Objetivo

Clone do famoso jogo [Flappy Bird](https://flappybird.io/), em PyGame.

### Descrição

O objetivo é desviar dos obstáculos e aguentar o máximo de tempo possível. O jogador tem apenas uma tecla que é para saltar.
